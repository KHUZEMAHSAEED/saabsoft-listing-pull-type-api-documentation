API Documentation

Base URL: https://youtupia.net/SingleListingApp/api

1. Client Credentials

Steps to Generate Client ID & Client Secret:

  1. Go to Website Settings from Settings menu.
  2. Click on Generate Token.
  3. You will get your Client ID and Client Secret.



2. Generate API Token

POST /auth/token

Header: Content-Type: application/json

Request Body:

{
  "client_id": "your-client-id",
  "client_secret": "your-client-secret-key"
}

Sample Screenshot:

Required Headers

All protected endpoints require these headers:

  • X-API-Token: your-api-token
  • X-Client-ID: your-client-id

Without these, access will be denied.

Note: API Token is valid for 2 hours.

3.Properties List API

GET /get-data-properties?order[0][column]=0&order[0][dir]=asc

Sample Screenshot:

Available Filters

Parameter Description
price Price range (min,max)
category R: Residential, S: Commercial
purpose R: Rent, S: Sale
unit_no Unit number
property_type Property type abbreviation (e.g., AM, PK)
owner Listing agent ID
developers Developer name
location Community name
sub_community Sub-community name
building Building name
listing_title Text in the listing title
bedrooms Number of bedrooms
listing_id Listing IDs (comma-separated reference numbers)
permit_number Permit number

Sorting & Pagination

Parameter Description / Values
Sort By unit_id, listing_status, listing_id, listing_title, size, property_type, price, Listing_Last_Modified_Date
order[0][column] Column index (0 to 6)
order[0][dir] Sort direction (asc or desc)
limit Number of results per page (up to 100)
page Page number (e.g., page=1, page=2)
